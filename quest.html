Important Questions

  • Data types supported by JavaScript?
    The data types supported by JavaScript are:
    • Undefined
    • Null
    • Boolean
    • String
    • Symbol
    • Number
    • Object
  • JavaScript is Compiled or Interpreted?
    JavaScript Engines in browser or Node JS interprets JavaScript Code. It is a lightweight, interpreted programming language. Example of JS engine is V8 engine in Chrome.
  • Difference between let and const?
    let and const both are ES6 feature of JS. Both have block {} scope. let is used for creating variables whereas const is used for creating constant.
    • const can only be assigned once and multiple assignment will result in error.
    • while declaring const, assigning value to it is mandatory. 
      const age;
      will throw error, we shall assign value to age like 
      const age = 20;
  • Difference between let and var?
    let is ES6 feature and sustitute for var. Both are used for declaring variables.
    • var has function scope means 
                                  function hello() {
                                for(var i = 0; i<=10; i++) {
                                    console.log("Hi");
                                }
                            
                                console.log(i);
                            }
      Here, although variable i is declared inside for block using var keyword and it will be available anywhere inside its parent function. So it will be available outside for block as well.
    • let has Block scope means 
                                  function hello() {
                                for(let i = 0; i<=10; i++) {
                                    console.log("Hi");
                                }
                            
                                console.log(i);
                            }
      Here, variable i is declared inside for block using let keyword and it will be available only inside for block. i.e. last line console.log(i) will result in error;
  • Is JavaScript Object Oriented language?
    Not exactly. JavaScript is actually Object based language. We can create Object, we can use object but we cannot create Classes like Object Oriented language. Means we cannot actually create blueprints for creating object. 
                            // to create object
                        let studentObj = {
                            name : 'John',
                            age : 20,
                            passed : true;
                        };

                        // to change property value
                        studentObj.name = 'Johnny';

                        //add new property
                        studentObj.city = 'Bangalore';

                        // access property
                        console.log(studentObj['age']);
                        console.log(studentObj.city); // alternate
